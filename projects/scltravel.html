Khám phá du lịch Đồng Bằng Sông Cửu Long

  • Backend: PHP Native
  • Database: MySQL
  • Frontend: HTML, CSS, JavaScript
  • Repo: GitHub Link

Mục tiêu

Xây dựng nền tảng thông tin du lịch toàn diện cho khu vực ĐBSCL. Hệ thống cho phép người dùng tìm kiếm địa điểm, xem bài viết review và admin quản lý nội dung dễ dàng.

Chức năng chính

  • Tìm kiếm & Lọc: Theo tỉnh thành, loại hình du lịch.
  • Quản trị (CMS): Thêm/Sửa/Xóa bài viết, quản lý User.
  • Tương tác: Đăng ký, Đăng nhập, Bình luận.
